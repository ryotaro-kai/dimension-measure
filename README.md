# Dimension Measure

WebXRを使用したAR計測アプリケーションのモック実装です。スマートフォンのカメラを通じて、AR空間に配置した3Dボックスのサイズをリアルタイムで計測できます。

## 機能

### AR床認識
- **Hit Test API** を使用して床平面を自動認識
- レティクル（リング型カーソル）で床検知位置を可視化
- ユーザーがタップして計測対象を配置

### 3Dボックス計測
- タップでAR空間にボックスを配置
- リアルタイムでサイズ変更が可能
- 幅・高さ・奥行きを独立して調整

### インタラクティブUI
- スライダーコントロール（幅: 5-100cm、高さ: 5-100cm、奥行き: 5-100cm）
- 現在のサイズ表示
- リセットボタン（ページリロード）

### 3D表現
- Three.jsを使用した3Dレンダリング
- 半透明ブルーのボックスメッシュ
- ワイヤーフレーム表示（白い枠線）
- HemisphereLight による自然な照明

## 技術スタック

- **WebXR Device API**: AR機能の提供
- **Three.js**: 3Dグラフィックスレンダリング
- **HTML5 Canvas**: レンダリング対象
- **JavaScript (ES Modules)**: モダンモジュール形式

## 使用方法

1. WebXR対応デバイス（AR対応スマートフォン）でアクセス
2. **「Enter AR」ボタンをタップ** してAR モードを起動
3. 床面が認識され、リティクルが表示される
4. リティクルの位置で **タップ** してボックスを配置
5. スライダーでサイズを調整
6. **リセットボタン** でアプリをリセット

## ブラウザ対応

- Android: Chrome (Android 7.0以上)
- iOS: Safari + WebXR対応（ARCore または ARKit）

## 備考

- 底面を基準にボックスが配置される（Y軸 +0.5オフセット）
- サイズ単位はセンチメートル（cm）
- 計測値のメートル変換は内部的に自動処理
